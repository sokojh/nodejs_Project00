# nvm 버전관리
https://github.com/coreybutler/nvm-windows/releases 에서 

nvm-setup.zip를 설치하면 터미널에 nvm키워드가 작동

nvm install 16.8.0
터미널에 입력해서 설치

nvm use 16.8.0
입력하고 동의하면 노드 버전 변경

버전 리스트 출력
nvm list available : 모든 리스트보기
nvm list : 자기 버전 보기
